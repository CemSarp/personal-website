Mobile Messaging App

Developed a mobile messaging app with full-stack implementation. Designed and built both frontend (React Native) and backend (MongoDB).

  • Built the frontend using React Native, ensuring a responsive and intuitive user interface tailored for iOS devices.
  • Designed and implemented a robust backend architecture with MongoDB, supporting scalable data storage and retrieval.
  • Integrated JWT-based user authentication and end-to-end encrypted messaging to ensure user privacy and data security.
  • Enabled real-time communication using WebSockets, providing seamless chat functionality and instant updates.
  • Deployed and tested the application on iOS platforms, validating performance, security, and usability.

Research Assistant – PURE Initiative, Sabancı University

Conducted a data-driven research project under Sabancı University’s PURE (Program for Undergraduate Research) initiative, focusing on Corporate Social Responsibility (CSR) communication strategies during the 2023 Kahramanmaraş earthquake.

  • Analyzed thousands of disaster-related posts from 60 local and international companies on social media (Platform X), using custom data scraping and manual annotation methodologies.
  • Classified brand communications into informational, transformational, and interactional types to assess public engagement patterns.
  • Evaluated the effectiveness of corporate giving, employee involvement, and technical support as CSR strategies during crises.
  • Identified emotional storytelling and multimedia content (especially images) as key drivers of engagement and trust in CSR communication.
  • Provided practical recommendations for improving disaster communication logistics and enhancing stakeholder trust through empathetic and interactive CSR messaging.
